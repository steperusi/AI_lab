# AI-Lab

A collection of AI algorithms and models implemented in Python, covering the core areas of modern artificial intelligence: search, planning, reinforcement learning, and machine learning.

Environments are built with OpenAI Gym; neural networks use Keras.

## Topics Covered

**Uninformed Search**

* Breadth-First Search (BFS)
* Depth-Limited Search (DLS)

**Informed Search**

* Uniform Cost Search (UCS)
* Greedy Best-First Search
* A*

**Markov Decision Processes**

* Value Iteration

**Reinforcement Learning**

* Q-Learning
* SARSA

**Machine Learning**

* Deep Neural Networks (DNN) with Keras

## Tech Stack

Python · OpenAI Gym · Keras · sklearn
